# Rent Bike App
## 1. Start app manual
## 1.1 Docker

Rename example.env to .env.
```sh
$ mv example.env .env
```

Build and up with Docker
```sh
$ docker-compose build
$ docker-compose up
```

Go to:
```sh
http://localhost:4000/
```
## 1.2 Local

Rename example.env to .env.
```sh
$ npm run local:prod
```

Go to:
```sh
http://localhost:4000/
```
